Analyse du problème d'alignement

Problème identifié :

  • Les icônes Bootstrap ont des largeurs variables intrinsèques
  • Même avec width: 24px, l'icône elle-même peut être plus étroite ou plus large visuellement
  • Le display: inline-flex avec justify-content: center centre l'icône dans son conteneur, mais ne garantit pas l'alignement visuel

Solution recommandée :

  • Utiliser display: grid avec grid-template-columns: 24px 1fr
  • Cela crée une colonne fixe de 24px pour l'icône et une colonne flexible pour le texte
  • L'alignement est garanti par la grille CSS
