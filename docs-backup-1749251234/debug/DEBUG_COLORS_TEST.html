Comparaison des couleurs

--tc-color-gray-50
hsl(202, 8%, 98%)
Gris très clair
--tc-color-primary-50
hsl(202, 36%, 96%)
Bleu très clair
--tc-color-primary-100
hsl(202, 36%, 89%)
Bleu clair

Note: La différence entre gray-50 (8% saturation) et primary-50 (36% saturation) devrait être visible - le primary-50 devrait avoir une teinte bleutée.
